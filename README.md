# Snakes Battle Game

## Development

### [1] Install all dependencies

In `client`

```sh
npm ci
```

In `server`

```sh
npm ci
```

### [2] Create `.env` file in `server` directory and paste this:

```sh
PORT=8080
ENV_MODE="local"
```

### [3] Run project

For `client`

```sh
npm run dev
```

For `server`

```sh
npm start
```

### [4] Got to `http://localhost:<PORT>` and have fun !
